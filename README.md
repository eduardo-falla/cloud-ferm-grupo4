# CLOUD-FERM – Sistema Cloud de Gestión de Fermentación de Cacao

## Descripción general

CLOUD-FERM es una aplicación web en la nube que permite **monitorear, controlar y trazar** el proceso de fermentación de cacao en la Finca Don Valentín.  
El sistema se apoya en servicios de Microsoft Azure y una base de datos en la nube para ofrecer:

- Monitoreo en tiempo real de variables críticas (temperatura, humedad, CO₂, COVs).
- Actuación remota sobre el proceso (comandos de mezclado, reinicio de dispositivo).
- Gestión de lotes y trazabilidad histórica mediante reportes y exportación de datos.

Este proyecto corresponde al curso **Capstone / Administración de Proyectos de Software**.
